# Getting Started

CyberWallet SDK is a collection of tools that allow developers to integrate CyberWallet into their dapps. The SDK provides a simple and secure way to connect to CyberWallet and interact with the Cyber mainnet.


## Supported Environments

CyberWallet SDK only supports dapps in the browser environment.

## Supported Networks

Mainnets:
* Cyber Mainnet

Testnets:
* Cyber Testnet

## Singature Validation with EIP-6492

If your dapp has features that needs to validate the signature like SIWE (Sign-In with Ethereum
), you'll needs to implement EIP-6492 to support that. You can find the EIP-6492 detail [here](https://eips.ethereum.org/EIPS/eip-6492).
